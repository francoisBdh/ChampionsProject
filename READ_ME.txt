
Module IDM :

Nous avons réalisé le projet Champions project qui  affiche les informations de différents ligues de football de différents pays.

Notre projet génère un page HTML avec des données.

Nous avons implémenter un fichier Gen.xtend qui permet de  via différentes fonctions toHTML() de générer un fichier html qui nous enregistrons dans notre projet
Le chemin : 

        ChampionsProject\championsProject.XText\test.html

fichier .cpl (notre extension correspondant) : test.cpl

chemin fichier .xtend : ChampionsProject\championsProject.XText\src\championsProject
